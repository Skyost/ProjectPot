Projectpot

Projectpot is just like a honeypot, but for your projects. It is a simple project repository written in PHP and it does not need any database (uses JSON).

Instructions

Download the latest release here and uncompress it on your server. That's all !

Note : The first time you use this software, the admin username is admin and the admin password is admin. It is strongly advised that you change this as quick as possible (check the file include\config.php) !
